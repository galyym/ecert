Это мой проект на Laravel: https://vsestroi.kz. 
Фронтенд написан на Blade (Laravel). Сейчас это одностраничный сайт с секциями, между которыми можно переходить по якорям. 

Задача:
1. Превратить сайт в современный корпоративный сайт с отдельными страницами/разделами (например: Главная, О компании, Услуги, Проекты, Новости, Контакты).
2. Обновить визуальную часть (UI/UX) — сделать современный и стильный дизайн (корпоративный, чистый, с акцентом на доверие и профессионализм).
3. Все формы, которые есть сейчас, должны остаться на фронте:
   - форма для подачи заявок на сертификацию,
   - форма для поиска сертификатов.
   Их функционал менять не нужно, только визуальное оформление.
4. Админка написана на Laravel Moonshine. Нужно:
   - при добавлении новых разделов (например, "Услуги", "Новости") сделать их редактируемыми через Moonshine,
   - предусмотреть возможность управления контентом (тексты, изображения, список услуг, новости).
5. Не нужно менять бэкенд-логику приёма заявок и поиска сертификатов — только фронтенд и интеграцию с Moonshine.
6. Использовать Blade для фронта. Код должен быть чистым и легко расширяемым.

Выведи полностью обновлённый фронтенд проекта (Blade-шаблоны, стили, структура страниц), а также необходимые изменения для Moonshine (ресурсы, модели) с учётом редактируемости новых разделов.
